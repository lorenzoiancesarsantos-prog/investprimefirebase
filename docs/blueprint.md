# **App Name**: Invento Simples

## Core Features:

- User Registration: Allows users to create an account by providing their full name, email, password, and credit card details, and accepting the user terms.
- Login Authentication: Authenticates users based on validated email and password credentials.
- Portfolio Overview: Generates a dashboard summarizing the user's investment status: total invested, monthly gains, royalty quantity, and available balance for withdrawals.
- Investment Evolution: Displays a chart that plots investment and return gains over time.
- Referral Program: Keeps track of the amount and value of referrals; supports up to 10 referrals per user.
- AI-Powered Investment Tool: LLM analyses historical financial data, simulates potential investment scenarios, and, using this 'tool', suggests optimal royalty purchase strategies based on individual investment objectives and risk profiles.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), conveying sophistication and trust.
- Background color: Light Lavender (#E6E6FA), offering a soft, clean backdrop.
- Accent color: Violet (#8F00FF), highlighting key interactive elements and calls to action.
- Headline font: 'Playfair', serif, to add an elegant, fashionable, high-end feel.
- Body font: 'PT Sans', sans-serif, paired with Playfair to allow for longer amounts of text.
- Crisp, professional icons representing investments, gains, and user profile sections.
- Smooth transitions and loading animations to provide a polished and professional user experience.
